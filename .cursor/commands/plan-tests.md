# /plan-tests

Create a comprehensive Playwright test plan for the Invoice Canvas app.

Use the `playwright-test-planner` skill. Explore the target screen or flow with the
`playwright-test` MCP and save a markdown plan to `specs/`.

**Argument:** the screen or flow to plan (e.g. `Dashboard`, `Create Invoice`,
`Invoice Overview`). If a persona is named (PM / BDU / Admin), include role-based scenarios
with expected allow/deny/visible/hidden outcomes.

Target: **$ARGUMENTS**
